# Aula 07 - 03/03/26

## Conteúdo da Aula

A aula foi mais teórica e centrou-se em conceitos de CSS:
- Utilização de `id` e `class` para seleção de elementos (`#` e `.`).
- Observação de que `id` deve ser único na página.
- Diferença entre `margin` e `padding`.
- Como inspecionar elementos e propriedades em sites usando ferramentas de desenvolvedor.

Além disso, foi passado um trabalho prático em HTML e CSS:
- Tema: **"O Alfaiate Voador"**, obra de Louis-Sebastien Lenormand.
- O conteúdo a ser utilizado (imagens e textos) está na pasta `Alfaiate_Voador_Trabalho` dentro desta aula.
- O layout é de livre organização, preferencialmente em **single‑page** (uma única página).
- Professor recomendou o site <https://www.iloveimg.com/pt> para redimensionar e tratar as imagens.

### Estrutura dos Arquivos

- `Alfaiate_Voador_Trabalho/`: contém os recursos fornecidos (imagens, textos) para o exercício.

### Navegação
- Todo o conteúdo estará em uma única página (`index.html`), seguindo o modelo de single‑page.

### Objetivos Didáticos
- Praticar a criação de seletores CSS com `id` e `class`.
- Entender a aplicação de `margin` e `padding` em elementos.
- Desenvolver um layout livre utilizando HTML e CSS com base no material fornecido.

---

> **Dica:** use as ferramentas de desenvolvedor do navegador para inspecionar elementos de sites e aprender com exemplos reais.
