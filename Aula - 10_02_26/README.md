# Aula 03 — 10/02/2026

## Resumo
Terceiro encontro: aprofundamento em listas HTML, incluindo listas ordenadas, não ordenadas e listas de definição. Estudo sobre aninhamento de listas e personalização de marcadores.

## Conteúdos trabalhados
- Listas ordenadas (`<ol>`) e não ordenadas (`<ul>`)
- Itens de lista (`<li>`)
- Aninhamento de listas (listas dentro de listas)
- Atributo `type` para personalizar marcadores (`disc`, `circle`, `square`)
- Listas de definição (`<dl>`, `<dt>`, `<dd>`)
- Tags semânticas: `<section>`, `<header>`, `<main>`, `<footer>`
- Hierarquia de títulos (`<h1>`, `<h2>`, `<h3>`)
- Boas práticas: fechamento de tags e estrutura semântica correta

## Arquivos
- `index.html`: página exemplo com diferentes tipos de listas (consoles de videogame e definições de tecnologias web).

## Como visualizar
- Abra `index.html` no navegador ou use o Live Server no VS Code.

## Extensões úteis no VS Code
- Live Server — servir páginas HTML localmente
- HTML CSS Support — autocomplete e dicas
- Prettier — formatação automática
- IntelliSense for CSS class names — sugestões de classes
