# Aula 05 - 24/02/26

## Conteúdo da Aula

Esta aula aborda a organização modular de um site didático de HTML e CSS, com navegação centralizada e exemplos práticos de cada tema. O objetivo é demonstrar boas práticas de estruturação, navegação e separação de conteúdo em páginas distintas.

### Estrutura dos Arquivos
- `index.html`: Menu principal da aula, com links para cada tópico abordado.
- `links.html`: Exemplos de links internos e externos.
- `downloads.html`: Exemplos de links para download de arquivos.
- `imagens.html`: Demonstração de imagens responsivas utilizando `<picture>` e media queries.
- `audio.html`: Exemplo de áudio com controles, autoplay e loop.
- `css-inline.html`: Demonstração de CSS inline (no atributo `style`).
- `css-interno.html`: Demonstração de CSS interno (dentro da tag `<style>` no `<head>`).
- `css-externo.html`: Demonstração de CSS externo (arquivo `styles.css`).
- `styles.css`: Folha de estilos externa utilizada nas páginas.
- `script.js`: Exemplo de uso de JavaScript externo.
- Pasta `Arquivos/Imagens`, `Arquivos/Audios`, `Arquivos/Documentos`: Contêm os arquivos utilizados nos exemplos (imagens, áudios, PDFs, etc).

### Navegação
- O menu principal está localizado em `index.html` e permite acesso rápido a todos os tópicos.
- Cada página temática possui um link de retorno para o menu principal.

### Objetivos Didáticos
- Demonstrar a separação de conteúdo por tópicos.
- Ensinar a navegação entre páginas HTML.
- Apresentar diferentes formas de aplicar CSS (inline, interno, externo).
- Exibir exemplos práticos de links, downloads, imagens responsivas e áudio.

### Observações
- Todos os caminhos de arquivos e imagens foram ajustados para refletir a estrutura de pastas do projeto.
- O arquivo `menu.html` foi removido, centralizando a navegação no `index.html`.
