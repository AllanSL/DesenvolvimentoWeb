# Aula 09 - 10/03/26

## Conteúdo da Aula

Nesta aula trabalhamos com Bootstrap e construção de layout responsivo:
- Inclusão do Bootstrap (CSS e JS) via CDN.
- Criação de um layout em grid com 3 colunas na mesma linha (telas grandes), 2 colunas sobre 1 (telas médias) e empilhadas em telas pequenas.
- Preenchimento dos blocos com texto Lorem Ipsum para demonstrar o comportamento responsivo.

### Arquivos principais
- `index.html`: contém o exemplo do grid responsivo (três colunas com conteúdo Lorem Ipsum).
- `styles.css` (opcional): estilos customizados se necessário.

### Objetivos Didáticos
- Entender o sistema de grid do Bootstrap e como definir colunas que mudam de comportamento conforme o breakpoint (col-*, col-md-*, col-lg-*).
- Preparar a base para exercícios de formulário e responsividade nas próximas aulas.
