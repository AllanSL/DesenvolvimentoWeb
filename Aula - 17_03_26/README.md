# Aula 11 - 17/03/26

## Conteúdo da Aula

Exercícios práticos de JavaScript:
- Manipulação de texto e DOM
- Implementação de um contador simples
- Validação básica de formulário (campo obrigatório)

### Estrutura dos Arquivos
- `index.html`: página com exercícios interativos.
- `script.js`: implementações dos exercícios em JavaScript.
- `styles.css`: estilos opcionais.

### Navegação
- Abra `index.html` no navegador (ou com Live Server) para testar os exercícios.

### Objetivos Didáticos
- Praticar seleção e manipulação de elementos do DOM.
- Trabalhar com eventos (click) e atualização de conteúdo dinâmico.

---

> **Observação:** Edite `script.js` para adicionar mais exercícios ou ajustar comportamentos.
