# Aula 06 - 02/03/26

## Conteúdo da Aula

Esta aula abordou:
- Estruturação de página com título, subtítulos e parágrafos.
- Utilização de uma imagem como título da página.
- Criação de um botão que, ao ser clicado, troca a cor de um parágrafo usando JavaScript.
- Organização dos arquivos em HTML, CSS e JS separados.

### Estrutura dos Arquivos
- `index.html`: Página principal da aula, com imagem de título, quatro subtítulos e parágrafos, botão para interação JS.
- `styles.css`: Folha de estilos externa, centralizando elementos e controlando o tamanho da imagem.
- `script.js`: Script JS com função para trocar a cor do parágrafo ao clicar no botão.
- Pasta `Imagens/`: Contém a imagem utilizada como título (`image.png`).

### Navegação
- O conteúdo está todo em `index.html`.
- O botão interativo está ao final da página, logo após o último parágrafo.

### Objetivos Didáticos
- Praticar a separação de arquivos HTML, CSS e JS.
- Demonstrar manipulação de elementos via JavaScript.
- Explorar a centralização e organização visual com CSS.

---

> **Dica:** Edite o valor de `width` no seletor `img` do CSS para ajustar o tamanho da imagem de título conforme desejar.
