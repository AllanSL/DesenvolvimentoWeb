# Aula 28/04/26 - Projeto Node.js

## Como rodar o projeto

1. Instale as dependências:
   ```bash
   npm install
   ```
2. Inicie o servidor:
   ```bash
   npm start
   # ou para desenvolvimento com reinício automático:
   npm run dev
   ```

O servidor rodará em http://localhost:3000
