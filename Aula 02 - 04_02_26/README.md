# Aula 02 — 04/02/2026

## Resumo
Segundo encontro: apresentação do elemento `<img>` e estudo sobre caminhos relativos e absolutos para referência de arquivos em HTML.

## Conteúdos trabalhados
- Elemento de imagem (`<img>`)
- Atributos: `src`, `alt`, `title`, `style`
- Caminhos relativos: implícito (sem `./`) e explícito (com `./`)
- Caminhos relativos para subpastas
- Caminhos absolutos (URLs da web)
- Organização de arquivos em pastas (estrutura de diretórios)

## Arquivos
- `index.html`: página exemplo com diferentes formas de referenciar imagens.
- `luan_melancia.jpg`, `ramilho.jpg`: imagens na mesma pasta do HTML.
- `Imagens/`: pasta contendo `jo77.jpg` e `gil_goiaba.jpg`.

## Como visualizar
- Abra `index.html` no navegador ou use o Live Server no VS Code.

## Extensões úteis no VS Code
- Live Server — servir páginas HTML localmente
- HTML CSS Support — autocomplete e dicas
- Prettier — formatação automática
- IntelliSense for CSS class names — sugestões de classes
