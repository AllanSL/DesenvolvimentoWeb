# Aula 08 - 04/03/26

## Conteúdo da Aula

Nesta aula trabalhamos com o framework **Bootstrap**:
- Importação e utilização dos arquivos CSS e JavaScript do Bootstrap.
- Uso do sistema de grid (linhas e colunas), incluindo colunas encadeadas, para modelagem do layout da página.
- Foi fornecido um esboço de um formulário que devemos replicar como exercício; usaremos esse mesmo layout na próxima aula para tratar de responsividade.

### Estrutura dos Arquivos
- `index.html`: página principal, já contendo um exemplo simples de grid com colunas e as classes do Bootstrap.
- `style.css`: arquivo de estilos adicional (apenas para bordas e centralização nos exemplos).

### Navegação
- Todo o conteúdo da prática está em `index.html`.
- O formulário de exemplo deverá ser copiado/replicado nesta mesma página.

### Objetivos Didáticos
- Aprender a incluir e usar o Bootstrap em projetos.
- Entender o funcionamento da grid do Bootstrap e como encadear linhas/colunas.
- Preparar um layout de formulário que servirá de base para exercícios de responsividade na próxima aula.

---

> **Observação:** a próxima aula continuará com o mesmo formulário, adicionando estilos responsivos e ajustes para diferentes tamanhos de tela.
